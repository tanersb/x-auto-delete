# X Auto Delete My Tweets

Deletes only your own tweets on X (Twitter).  
Works with a manual Start / Stop button. No automatic actions.

---

## Features

- Deletes **only tweets that belong to your account**
- Never touches other usersâ€™ tweets
- Manual control with a floating Start / Stop button
- Prevents double-processing
- Scrolls down and continues automatically

---

## Installation

1. Install **Tampermonkey**
2. Create a new userscript
3. Paste the contents of `x-auto-delete.user.js`
4. Save

Or install directly:

https://raw.githubusercontent.com/tanersb/x-auto-delete/main/x-auto-delete.user.js


---

## Usage

1. Open your X (Twitter) profile
2. A blue button appears on the right side of the page
3. Click **START DELETE** to begin
4. Click **STOP DELETE** to pause

Deletion only happens while the script is running.

---

## Safety

- If the **Delete** option does not exist, the tweet is skipped
- No blocking, reporting, or interaction with other accounts
- Only your own tweets are affected
